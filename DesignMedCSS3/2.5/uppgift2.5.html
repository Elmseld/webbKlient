a:hover

Exempel på länk vid hovering

cursor

Sätter vilken typ av muspekare som ska synas när man för musen över ett element, ett exempel;

	cursor: help;

Skulle se ut såhär(håll muspekaren över denna text)!

Länk som en knapp

Att styla en länk som en knapp kan man göra genom att ge den en border och en bakgrundsfärg, ett exempel;

a#button:link {
	border: 1px solid;
	background-color: cadetblue;
}

a#button:visited {
	background-color: aqua;
}

a#button:hover {
	background-color: darksalmon;
}

a#button:active {
	border: 2px dotted;
}

Skulle bli såhär;

Länk som en bild

Att göra en länk med grafik gör du genom att lägga till en bakgrundsbild på en länk, tex såhär;

a#img:link {
	border: 1px solid;
	padding: 5px 5px 5px 1.7em;
	background-image: url(../img/folderCheck.png);
	background-repeat: no-repeat;
}

a#img:visited {
	color: cornflowerblue;
}

a#img:hover {
	background-color: crimson;
}

a#img:active {
	border: none;
}

Skulle bli såhär;
