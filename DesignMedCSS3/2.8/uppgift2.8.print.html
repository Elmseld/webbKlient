Uppgift 2.8 CSS-konstruktioner för skrivare

@page

Används för att ange layouten på en page box, boxen består av två delar en sid-del och en marginal-del, det finns tre olika psudo-selectorer(:first, :left, :right) man kan använda för första sidan, höger och vänster sida, man kan inte använda sig av px eller pt då en skrivare inte förstår detta utan måste använda mått som tex cm. Ett exempel kan se ut såhär i kod;

@page {
	margin: 3cm;
}

page-brake-after

Sätter om en sidbrytning borde ske efter elementet, exempel i kod;

footer {
	page-brake-after: always;
}

Denna kod sätter att en sidbrytning ska se efter varje footer och finns sidmallen för utskrift av denna sida.

page-brake-before

Sätter om en sidbrytning ska ske före ett element, exempel i kod;

h2 {
	page-brake-before: avoid;
}

Denna kod sätter att om möjligt undvika att sidbryta före bilder och finns sidmallen för utskrift av denna sida.

page-brake-inside

Sätter om en sidbryting är tilllåten inom ett element, ett exempel i kod;

article {
	page-break-inside: avoid;
}

Denna kod sätter att om möjligt undvika att sidbryta inom artikel element och finns sidmallen för utskrift av denna sida. Ingen av page-break propertys kan användas på absolut positionerade element.

size

Sätter storleken på sid-delen av page boxen, ett exempel i kod;

@page {
	size: 8.5in 11in; 
}

orphans

Bestämmer de minsta antalet rader som måste lämnas på botten av en sidan när en sidbrytning sker inom ett element, ett exempel i kod kan se ut såhär;

orphans: 3;

Denna kod finns med i sidans stilmall vid utskrift.

widows

Bestämmer de minsta antalet rader som måste lämnas på toppen av en sidan när en sidbrytning sker inom ett element, ett exempel i kod kan se ut såhär;

widows: 3;

Denna kod finns med i sidans stilmall vid utskrift.
