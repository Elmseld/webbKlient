Webbsideslayout (fixerad, flytande och elastisk) (fixed, fluid, and elastic)

Det finns olika begrepp för ur man kan bygga upp en webbsideslayout, en fixerad layout skapas genom att man använder fasta mått för att positionera de olika elementen på en hemsida, detta gör att hemsidan alltid ser likadan ut oberoende av vilken enhet man använder, detta kan dock gör att använder på små enheter, tex. smartphones måste scrolla horizontelt för att se hela innehållet vilket inte blir så användarvänligt. En fördel är däremot att designen på en hemsida kommer alltid se ut som man tänkt oberoende på enhet.

Med en flytande layout däremot använder man istället procentuella mått när man placerar ut elementen på sidan vilket gör att de anpassar sig efter olika storlek på användarnas skärmar, detta gör den mer användarvänlig men också svårare att hålla koll på hur designen kommer att se ut på olika skärmar.

Man kan också använda en elastisk layout, vilket är lite av en blandning mellan de två föregångarna, där alla mått sätts med em, vilket gör att storleken på hemsidan anpassas efter textstorleken men inte efter skärmstorleken, denna layout kan vara väldigt användarvänlig men kan också leda till mer extra arbete men testning etc. vilket inte alltid kan kännas värt det.

Varför man ska använda CSS och inte tabeller

Tabeller är inte gjorde för att skapa en webbsideslayout så försöker man göra de ändå leder det till mycket extra arbete, extra mycket och plottrig htmlkod som senare kan bli besvärlig att modifiera.

Position

position

Specifierar typen av postions metod som används för att positionera ett element, tex, static, relative, absolute eller fixed. ett exempel skulle se ut såhär i kod;

position: absolute;

top, left, bottom, right

Dessa propertys sätter alla avståndet till antingen element runt sig när propertyn position har attributet absolute och i relation till sin naturliga utgångspunkt om position har attributet relative, top för topmåttet, left för vänstermåttet, bottom för bottenplacering och right för högerplacering. Ett exempel kan se ut såhär i kod;

position: absolute;
		top: 710px;
		left: 30px;

z-index

z-index sätter vilket element som ska vara framför ett annat om båda elementen ligger på samma eller delvis samma plats, den med högst z-index ligger främst, detta genom att ange dess position i stacken. Ett exmepel i kod;

img.zhigh {
	z-index: 1;
}

img.zlow {
	position: relative;
	right: 25px;
	z-index: -1;
}

ser ut såhär;

Då hjärtat(zhigh) har ett högre z-index än den andra bilden ligger de över.

float

float bestämmer om ett element ska flyta eller inte och i så fall till höger eller vänster, så här kan ett exempel se ut i kod;

img.floatex  {
    float: right;
}

Skulle se ut såhär;

clear

Med clear sätter man om man inte vill att element ska flyta runt ett flytande element på en eller båda dess sidor, ett exempel i kod kan se ut såhär;

p, h2, h3  {
    clear: both;
}

Detta exempel används på bilden ovan för att inte detta stycke ska ligga till vänster om bilden.

Storlek

width

Denna property används för att sätta bredden på ett element, detta dock utan padding, border och margin det är bara elementet i mitten som anges av width. max-width och min-width åtsidosätter width propertyn. Ett exempel på hur width kan användas i kod;

width: 140px;

height

Denna property används för att sätta höjden på ett element, detta dock utan padding, border och margin det är bara elementet i mitten som anges av height. max-height och min-height åtsidosätter height propertyn. Ett exempel på hur height kan användas i kod;

height: 140px;

max-width

max-height

Denna property sätter max höjden för ett element, ett element kan aldrig bli högre än max-height, ett exempel i kod;

min-width

Denna property sätter minsta bredden för ett element, ett element kan aldrig bli smalare än min-width, denna propert åtsidosätter både width och max-width, ett exempel i kod;

min-height

vertical-align

Denna property justerar om man vill sänka eller höja ett element vertikalt, eller om man vil att olikstora element ska följas åt i tex under(standard) eller överkant, ett exempel när de följs åt i överkant;

vertical-align: text-top;

Detta hela raden att följa överlinjen istället för den undre som är standard.
